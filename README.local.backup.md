# Gemelo Digital Rosario

Visual 3D/4D digital twin for Rosario, Gran Rosario, Paraná corridor and Villa Constitución.

## Product name

CityPulse 4D / Gemelo Vivo

## Scope

- Rosario
- Gran Rosario
- Paraná corridor
- Villa Constitución

## Stack

- Frontend: MapLibre GL JS + deck.gl
- Future visual layer: Three.js / WebXR
- Database: Postgres
- Automation: n8n
- Governance: SpecBridge

## Public data sources

- OpenStreetMap / Overpass
- Open-Meteo
- USGS GeoJSON feeds
- NASA FIRMS, where access policy allows

## Core principle

No fake real-time data. Every visible layer must expose source, freshness and status.
