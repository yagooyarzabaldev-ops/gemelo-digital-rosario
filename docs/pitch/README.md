# Paquete comercial — Gemelo Digital Rosario (v0.1.0)

Material para presentar la demo a municipios, puertos, industria, sponsors y partners
institucionales. Es un **paquete comercial**: no incluye código funcional.

| Documento | Para qué sirve |
|---|---|
| [one-pager.md](one-pager.md) | Resumen comercial de una página: qué es, qué problema resuelve, para quién, qué muestra la demo, por qué es seguro y próximo paso. |
| [demo-script.md](demo-script.md) | Guion de demo en vivo de 5–7 minutos, con tiempos, frases ancla y qué **no** prometer. |
| [commercial-positioning.md](commercial-positioning.md) | Posicionamiento por segmento (municipio, puerto/logística, industria, sponsor, partner institucional): dolor, valor, ángulo y próximo paso pago. |
| [roadmap.md](roadmap.md) | Roadmap incremental desde la demo estática hasta PersonaTwin real (condicionado a marco legal y proveedor autorizado). |
| [risk-and-privacy-notes.md](risk-and-privacy-notes.md) | Nota breve de riesgo y privacidad para acompañar la demo. |
